# Dolphin – Startup Operating System

**Dolphin** is a structured platform that helps founders validate ideas, track progress through clear milestones, connect with vetted service providers, and attract investors by showcasing de-risked startups.

Built with **Node.js + Express + MongoDB** on the backend and pure **HTML/CSS/JS** on the frontend.

## Features

- Role-based access: Founder, Investor, Provider
- 7-stage validation roadmap with enforced gating
- Milestone completion tracking + admin verification
- Contextual provider matching based on current startup stage
- Introduction requests from founders to providers
- Audit trails & basic analytics for founders
- Investor view of validated startups (score ≥ 70%)

## Tech Stack

**Backend**
- Node.js + Express
- MongoDB (Mongoose)
- JWT authentication
- bcrypt for password hashing

**Frontend**
- Vanilla HTML5 + CSS3 (Inter font from Google Fonts)
- Pure JavaScript (no frameworks yet)
- Fetch API for backend communication

**Planned / Future**
- Admin panel for milestone verification & provider approval
- Email notifications (nodemailer / SendGrid)
- Real-time updates (Socket.io or polling)
- React/Vite frontend migration
